MarkDown文本写作的的进阶（进阶学习）

发表于 2019-08-09 更新于 2019-10-15 分类于 博客搭建   |   字数统计: 588(字)   |   阅读时长≈ 2(分)

一、颜色、字体和大小相关设置

Ⅰ.文字的颜色、大小、字体设置：

  • <font color=填写颜色 size=填写大小 face=填写字体名字>书写内容</font>

  • 内容可以调换顺序

    • SIZE的大小浏览器默认为3，我们可以1-7选择

  • 不需修改的东西可以不写

    • 例：<font color=填写颜色>书写内容</font>只更改颜色
      • 字体一般用不到，但是可以改成
    • 宋体
    • 仿宋
    • 黑体
    • 楷体
    • 微软正黑体
    • 等等，百度HTML字体查询即可。

Ⅱ.文字背景的颜色设置

  • <table><tr><td bgcolor=#bfe1f1>文本内容</td></tr></table>
    • 样例：
背景色是：#bfe1f1
-
该编辑只能用在一行的开头

Ⅲ.颜色代码

  • 颜色代码
  • 常用颜色可以直接打英文：
    • red
    • gree
    • green
    • blue
    • orange
    • yellow
    • grey

Ⅳ.文字居中位置设置

二、上标和下标书写

Ⅰ.上标

  • <sup>内容</SUP>
    • 样例：232

Ⅱ.下标

  • <SUB>内容</SUB>
    • 样例：X2

三、导出文件

Ⅰ.导出

通过typora的文件选项中的导出，我们可以导出下列文件

  • 图像
  • html
  • PDF
  • word文件需要下载使用 pandoc,我们可以不直接使用pandoc，安装后就可以直接使用typora进行转换了，安装后不行的话就重启机器即可

四、插入图片的排版

Ⅰ.插入原图片

![](图片地址)

一般直接使用图床复制黏贴即可

Ⅱ.修改图片大小

  • <img width='宽度' height='长度' src="图片网址">

  • 根据比例：

    <img src="图片地址" width="60%">

Ⅲ.修改图片位置（最终渲染网页文件根据主题变化，若不能变化可具体查找主题文件>）

  • <img width = "宽度" height ="长度" src ="图片网址" align="位置">

    • 位置包括
      • 居中：center
      • 居左：left
      • 居右：right

Ⅳ.图片并排放置（最终渲染网页文件根据主题变化，若不能变化可具体查找主题文件>）
