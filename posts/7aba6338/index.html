Ⅱ、使用技巧

1.快捷键使用

  • 工作窗口全屏非全屏切换 — ctrl+m

  • 当前行首：Home

  • 当前行尾：End

  • 选择当前行：Shift+Home+End

  • 行注释 /取消行注释— ctrl+/ — 需选中要注释的代码

  • 块注释 — ctrl+shift+/ — 需选中要块注释的代码

  • 取消块注释 —ctrl+shift+\ — 需选中已被块注释的代码

  • 快速跳转到某一行 — ctrl+l — 需要手动输入行数

  • 查看继承关系 — ctrl+t

  • 小写字母变大写 — ctrl+shift+x — 需选中字符.

  • 大写字母变小写 — ctrl+shift+y — 需选中字符.

  • 移动当前行到下一行 — alt+↓ — 当前行指光标所在行, 无需选中当前行. 可移动多行, 需要选中要移动的行

  • 移动当前行到上一行 — alt+↑ — 当前行指光标所在行, 无需选中当前行. 可移动多行, 需要选中要移动的行

  • 复制当前行到下一行 — ctrl+alt+↓ — 当前行指光标所在行, 无需选中当前行. 可复制多行, 需要选中要移动的行

  • 复制当前行到上一行 — ctrl+alt+↑ — 当前行指光标所在行, 无需选中当前行. 可复制多行, 需要选中要移动的行

  • 回到上次编辑点 — ctrl+q — 在脑袋不好使的时候强烈推荐这个

  • 删除当前行 — ctrl+d — 无需选中当前行, 光标在就行

  • 切换不同窗口—ctrl+F6

  • 自动格式化代码—ctrl+shift+f

    如果快捷键失灵，请检查搜狗或者其他输入法快捷键干扰问题，搜狗则打开高级快捷键设置，将简繁体快捷键取消即可

  • 放大或者缩小字体—ctrl+shift++则放大-则缩小

  • 折叠当前类中的所有代码—Ctrl+/(小键盘)

  • 展开当前类中的所有代码—Ctrl+×(小键盘)

  • 重构的后悔药—Alt+Shift+Z

  • 自动引入所需要得包—*Ctril+Shift+O *

  • 查找所需要的包—*Ctril+Shift+M *

  • 重命名—Alt+Shift+R

  • 把一段函数内的代码抽取成方法—*Alt+Shift+M *

具体使用

改变整体主题

改变整体theme：Window—>Preferences—>General—>Apperance选择主题为Dark。否则只是改变代码区主题。

获取更多代码区主题，下载对应EPF文件，导入即可。

导入方法：file->import->General->preferences->from preference file选择下载的epf文件，点击finish即可使用好看的主题。建议新建一个文件夹专门存放epf文件，这样每次去找就很方便。

也可以直接下载插件：https://pan.baidu.com/s/1JxEB3YCuH827Y3f-7th7cw#list/path=%2F

下载后复制到eclipse安装目录下的dropins文件夹下，在window>preferences>General>Appearance下会有二十多个主题可供选用。

自定义代码风格

Window—>Preferences—>Java—>Code Style—>Formatter。

Active profile中显示的是当前正在使用的代码自动格式化配置。点击“Export All…”按钮可以导出Eclipse中的代码自动格式化配置，点击“Import…”按钮可以导入本地已有的配置文件，导入和导出的文件都是xml格式。可以点击“New…”按钮，在Eclipse中的一个已有配置基础上，新建一个自己的配置。

详细请查看https://blog.csdn.net/jmyue/article/details/11060003

个人只是在Eclipse[built in]下新建风格，更改了Braces positions选项，全部改为Next line。

更改后再使用自动格式化快捷键会根据你的风格进行格式化

代码自动补全

在Eclipse中，从Window -> preferences -> Java -> Editor -> Content assist -> Auto-Activation下，我们可以在”.”号后面加入我们需要自动提示的首字幕，比如”abc”，可以把26个字全加上：

.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ_
